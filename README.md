# Stochastic Modeling for Hospital Systems (Birth–Death Process)

## 📌 Project Overview
This project models **hospital patient flow** using a **Birth–Death stochastic process** to analyze system behavior under uncertainty.

It simulates:
- Patient arrivals (birth rate λ)
- Patient discharge/service (death rate μ)

to study:
- Congestion
- Wait times
- Resource utilization

---

## ⚙️ Key Features
- Simulation using **Gillespie algorithm**
- Visualization of **patient flow trajectories**
- Comparison of **empirical vs analytical distributions**
- Configurable parameters (λ, μ, time, simulations)
- Interactive **Flask web interface**

---

## 🧠 Concepts Used
- Birth–Death Process  
- Stochastic Modeling  
- Markov Chains  
- Probability Distributions  
- Gillespie Simulation
